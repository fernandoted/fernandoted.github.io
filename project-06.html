    Calçados

    Este projeto foi interessante pois se tratava de uma loja de calcados, porém sem e-commerce. Por isso foi adotada uma linha mais limpa no layout, parecida com catálogos, para que não se corra o risco de o visitante interpretar como venda e se frustrar. Usei Wordpress com alguns bons plugins e bastante customização.

    This project was interesting because was about a shoe store, but with no e-commerce. That is why I adopt a clean layout, like catalogs, avoiding that the visitant think that is a e-commerce website and get frustrated.
